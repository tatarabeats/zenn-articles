## なぜ作ったか

**毎朝2つのアプリを開くのが嫌だった。**

SBI証券と住信SBIネット銀行。この2つを行き来しないと自分の総資産が分からない。しかも投資信託と米国株の個別株を両方持っていると、SBI証券の保有資産欄にちゃんと表示されないことがある。

「今、自分は**全部で**いくら持ってるんだ？」という一番シンプルな疑問に、すぐ答えられない。それがずっとストレスだった。

マネーフォワードは試したことがある。でも無料版はリアルタイム反映されないし、有料版は月額課金が発生する。**資産管理のためだけにサブスクを1つ増やすのが嫌だった。**

それに、UIがダサい。

「だったら自分で作ればいい」と思った瞬間、Claude Codeがあることを思い出した。

## どうやって作ったか

Claude Codeに指示を出した。それだけ。

技術スタックは Claude が決めた：

- **Next.js + TypeScript**（フロント）
- **Turso（クラウドSQLite）+ Drizzle ORM**（DB）
- **Tailwind CSS v4 + shadcn/ui**（デザイン）
- **Vercel**（デプロイ）
- **Auth.js v5 + Google OAuth**（認証）

自分がやったのは「こういうデザインにしたい」「このカードの色を変えて」「アニメーションを滑らかにして」という指示を出すことだけ。**コードは1行も書いていない。**

機能実装のコストが圧倒的に低かったので、開発期間の大半を**見た目と使い心地の改良に集中できた**。
