## なぜ作ったか

SBI証券の円グラフは大雑把で、どの銘柄をいくら持っているかをグラフ上で確認できません。銀行残高まで含めると、複数のアプリを行き来しないと全体が見えない状態でした。

「**これさえ開けば全部わかる**」というものが欲しかったんです。既存サービスも試しましたが、UIを自分好みにしたくて自作することにしました。

---

## どうやって作ったか

Claude Codeに指示を出しました。それだけです！

- **Next.js + TypeScript**（フロント）
- **Turso（クラウドSQLite）+ Drizzle ORM**（DB）
- **Tailwind CSS v4 + shadcn/ui**（デザイン）
- **Vercel**（デプロイ）
- **Auth.js v5 + Google OAuth**（認証）

「このカードの色を変えて」「アニメーションを滑らかにして」と指示するだけで、あとはClaudeが全部やってくれます。ファイルの整理も、方針決定も、エラーの修正も。**作業中にYouTubeを見ていられます。** 本当に。

## コストは？

自分が使ったのは**Claude Max 20x（$200/月 / 約30,000円）**です。このアプリだけなら**Max 5x（$100/月 / 約15,000円）で十分**でした。複数のプロジェクトを同時に動かしていたので20xにしていただけです。
